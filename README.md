# my-webshop
webshop for my silversmithing project

bellpepper.se

## Languages and techniques:
- Next.js (frontend)
- Express (server)
- Sanity.io (cms/backend)
- MobX (state handler)
- Styled-components (styling)
- Sendgrid (email)

- Heroku (hosting)
- CloudFlare (enable https)


## Get started:
`npm run build`

`npm i && npm run dev`
